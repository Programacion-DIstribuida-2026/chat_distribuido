Chat en tiempo real

Los usuarios usan ObjectId de MongoDB (24 caracteres hexadecimales). Crea usuarios en /docs (POST /usuarios), copia el campo id de la respuesta y pégalo en “Mi ID” y “Compañero” según corresponda.

Otro dispositivo en tu red: no uses 127.0.0.1 en su navegador (eso es “su propio equipo”). La otra persona debe entrar con la IP local de quien ejecuta el servidor, por ejemplo http://192.168.0.15:9000/chat (la IP la ves con ip a, hostname -I o en Windows ipconfig). El servidor debe arrancar con uvicorn ... --host 0.0.0.0 y el firewall debe permitir el puerto 9000.
